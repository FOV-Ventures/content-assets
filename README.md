# FOV Ventures — Content Assets

Public marketing assets for FOV Ventures social media (LinkedIn, X), newsletters, and other public surfaces. Served via `raw.githubusercontent.com` so external tools (Buffer, Beehiiv, etc.) can fetch images by URL.

## Naming convention

```
YYYY-MM-DD-<short-slug>.<ext>
```

Examples:
- `2026-05-28-deep-tech-years-to-exit.png` — Thursday LinkedIn stat card
- `2026-05-26-reinventing-reality-hero.png` — David's blog launch image

## Raw URL pattern

```
https://raw.githubusercontent.com/FOV-Ventures/content-assets/main/<filename>
```

This is the URL to pass to Buffer / external CDNs.

## Notes

- Keep this repo public — `raw.githubusercontent.com` only serves files from public repos.
- Optimize images before committing (PNG/JPEG, ideally < 1MB).
- Don't commit unreleased or embargoed material.
